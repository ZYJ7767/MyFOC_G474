# MyFOC_G474
ZYJ 
STM32G474VET6_FOC_SMO_PLL

Version 1.0:
基于个人学习，与项目负载率要求，手写代码完成FOC基本算法、控制器、观测器；实现SMO+PLL的无感双闭环控制；
其中为方便调用，封装好了McM功能函数并给出接口： FOC算法模块，VF强拖，IF强拖，PID控制器，电流环速度环，SMOAtan，SMOPLL，Blend无感加权切换，斜坡加减速等函数

Version 2.0：
由于中断函数超时这一致命错误，导致上一般实际控制频率下降
这次修改新增了DWT时间戳功能接口，优化运算，大幅度减少中断耗时
整体无感驱动负载率达到30%，为后续ASW应用层与通讯提供基础
    
    Version 2.1；
    优化编译，改为使用dsp计算三角函数，负载率降为20%一下，为后续添加任务功能以及通讯功能留出空间，优化了观测器PLL带宽以及滑膜参数K
    Version 2.2
    修改了PLL功能函数，形成归一化，纠正了环路带宽，保守设置：电流环200Hz--PLL100Hz--速度环20hz

Version 3.0
由于本人现在使用的电机相电流畸变严重，存在5次7次谐波，此版本加入了比例积分谐振控制器，并附上接口于FOC_Functiuon.h，作用于FOC电流环，可明显抑制谐波，得到光滑的正弦波形，其中Wc设为20rad/s，Kr为谐振增益，可更具电机不同调整
